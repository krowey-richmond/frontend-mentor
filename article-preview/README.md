

# Article Preview Component

A responsive article preview card with a dynamic share tooltip.
Built as part of a Frontend Mentor challenge to practice layout structuring and interactive UI components.

---

## Overview

This project focuses on:

* Responsive layout (mobile-first)
* CSS Grid + Flexbox combination
* Share button interaction with tooltip behavior
* Clean component separation and scalable CSS structure

On mobile, the layout stacks vertically.
On desktop, the image and content align side-by-side using CSS Grid.

---

## Features

* Responsive design
* Interactive share button
* Tooltip animation
* Modern CSS architecture using `@layer`
* Accessible button with `aria-label`

---

## Built With

* HTML5
* CSS3 (Flexbox + Grid)
* Mobile-first workflow
* Vanilla JavaScript

---

## What I Learned

* How to combine Grid and Flexbox effectively
* Absolute positioning for tooltips
* Managing UI state with class toggling
* Organizing CSS using layers for clarity and scalability
